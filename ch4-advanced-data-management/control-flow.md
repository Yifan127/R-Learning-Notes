### Control Flow

* Loop
    * **for(var in seq) statement**
    * **while(cond) statement**
* Condition
    * **if(cond) statement**
    * **if(cond) statement1 else statement2**
    * **ifelse(cond,statement1,statement2)**
    * **switch(expr, statement)**
* User-Written **Function**
```
myfunction <- function(arg1,arg2){
      statements
      return(object)
  }
```
